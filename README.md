吾尝终日而思矣，不如须臾之所学也。是曰：学不可以已，学而时习之。

> 孙莘老识欧阳文忠公，尝乘间从，以文字问之，云：“无它术，唯勤读书而多为之，自工。世人患作文字少，又懒读书，每一篇出，即求过人，如此少有至者。疵病不必待人指摘，多作自能见之。”

常言道：勤读，长练，久自工。

望君自勉。
